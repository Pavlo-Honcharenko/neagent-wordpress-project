## üìù Project Description

This project is a full-featured real estate classifieds platform built with WordPress and the ListingHive theme.

The website includes:

- Real estate listings from agencies across Ukraine
- Automated mass import of listings from multiple XML feeds
- Structured taxonomy system for cities, regions, and property types
- 5,000+ SEO-focused articles across construction and real estate categories

Article categories include:

- Real Estate
- Construction Equipment
- Building Materials
- Construction & Renovation
- Interior Design
- Electrical Works
- Landscape Design
- Furniture & Home Appliances
- Security Systems
- Transportation Services
- Plumbing
- Garden Equipment
- Other related sectors

Approximately 90% of the article content was written by me, ensuring structured SEO growth and topical authority.

## Technical Responsibilities

From a technical perspective, I handled:

- Full front-end development (HTML / CSS / JS)
- WordPress integration
- HivePress and ListingHive configuration
- XML feed parsing and automated listing imports
- Custom Gutenberg blocks development
- Complete SEO optimization
- Performance optimization
- Security configuration
- Hosting migration
- Ongoing administration and maintenance

For this project, I intentionally avoided ACF and built page structures entirely with native Gutenberg, extending it with custom blocks when required. This approach ensured better long-term maintainability and reduced plugin dependency.

Basic accessibility improvements were implemented following WCAG principles, without using a dedicated accessibility plugin.
